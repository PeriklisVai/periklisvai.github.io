EUDI Wallet - Master thesis

Development of an Android application for digital wallets, compliant with the eIDAS regulation, using Java and Kotlin

Android Java Kotlin
2025

This project was carried out as part of my Master's thesis in the MSc program “Information Systems and Services” at the University of Piraeus. Its focus was the design and implementation of a prototype European Digital Identity Wallet (EUDI Wallet) for Android, aligned with the eIDAS 2.0 regulation.

The application was built on top of the official wallet-core library which was customized to support issuance and presentation of Verifiable Credentials using the OpenID4VCI and OpenID4VP standards.

I also developed a companion authentication app that utilized the EUDI Web Verifier Endpoint library, deployed in a Docker container. This setup enabled end-to-end scenarios where a user could register or log in by presenting credentials, either via QR code scanning or deep link flows.

The thesis highlighted the lifecycle of Verifiable Credentials, challenges in remote presentation flows, and the practical application of the EUDI Wallet architecture. The implemented use cases showcase how trusted digital credentials can enhance security, interoperability, and privacy in online services.

Ratings Notebook - Android app

Create ratings and organize them by using your own custom categories and tags.

Unity C# Visual Studio Android Blender
2022

Ratings Notebook is an Android application published on the Google Play Store that allows users to create and manage personal ratings. Whether it’s movies, books, restaurants, or any custom category, users can quickly add entries, assign scores, and keep their evaluations organized in one place.

The app provides a clean and intuitive interface with support for search, custom tags and categories and search filtering making it easy to revisit past ratings

Developed entirely in Java with a local SQLite database for offline persistence.

I developed the application as a fun project in my free time

Sky Legends - 3D Video game

Dissertation on Developing a 3D video-game for android platforms using Unity Game Engine

Unity C# Visual Studio Android Blender
2020

Sky Legends is a 3D aerial combat game developed as part of my dissertation project at the University of Piraeus. Built with Unity and Blender, the game delivers an immersive dogfighting experience on mobile devices.

The gameplay combines both single-player and multiplayer modes. In single-player, the opponent is powered by AI with behaviors such as patrolling, attacking, and evading. In multiplayer, players can engage in real-time dogfights over a local network, testing their skills against friends in intense aerial battles.

My work covered the full pipeline: 3D modeling and animation in Blender, scripting core mechanics in C#, designing UI and effects, and optimizing performance for mobile deployment.
